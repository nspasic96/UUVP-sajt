  1. Napisati na programskom jeziku C program za raèunanje faktorijela prirodog broja, na osnovu formule

    n! = 1 · 2 · … · n 

    			
long fact(int n)
{
	if (n == 1) return 1;
	else return n * fact(n - 1);
}
    Poziv: 
    			
void main()
{
	for (int i = 1; i < 9; i++) {
		printf("%ld\n", fact(i));
	}
}
